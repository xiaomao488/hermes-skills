### 3️⃣ 创建 Release（3分钟）

**直接访问：** https://github.com/xiaomao488/hermes-skills/releases/new

1. **Choose a tag:** 选择 `v1.0.0`
2. **Release title:** 输入 `v1.0.0 - 首次发布`
3. **Describe this release:** 复制下面的内容

```markdown
# 🎉 Hermes Skills 首次发布

## ✨ 包含的技能

### 📚 Web Documentation Reader
从网页中提取和学习技术文档内容。

**功能：**
- 智能提取网页正文内容
- 支持 Markdown 和 JSON 格式
- 自动提取代码块和链接
- 保存到 Obsidian 知识库

### 🔍 GitHub Project Learner
深入学习 GitHub 项目的完整内容。

**功能：**
- 提取 README、文档、代码结构
- 分析 Issues 和 Discussions
- 读取 Wiki 内容
- 支持批量学习多个项目

## 📦 安装

```bash
git clone https://github.com/xiaomao488/hermes-skills.git
cd hermes-skills
pip install -r requirements.txt
```

## 📖 文档

- [README](https://github.com/xiaomao488/hermes-skills#readme)
- [使用示例](https://github.com/xiaomao488/hermes-skills/blob/main/EXAMPLES.md)
- [贡献指南](https://github.com/xiaomao488/hermes-skills/blob/main/CONTRIBUTING.md)

## 🤝 贡献

欢迎提交 Issue 和 Pull Request！

## 📝 依赖

- Python 3.7+
- requests
- beautifulsoup4
- lxml
```

4. 勾选 ✅ **Set as the latest release**
5. 点击 **Publish release**
