# 🎉 Welcome to Hermes Skills!

Thank you for your interest in contributing to Hermes Skills! This project aims to make it easier for Hermes Agent users to learn from web documentation and GitHub projects.

## 🌟 What is Hermes Skills?

Hermes Skills is a collection of reusable skills for [Hermes Agent](https://github.com/NousResearch/hermes-agent) that enable:

- **Automated learning** from technical documentation websites
- **Deep analysis** of GitHub projects and repositories
- **Knowledge base building** in Markdown format (Obsidian-compatible)
- **Batch processing** of multiple documentation sources

## 🚀 Getting Started

### For Users

1. **Install the skills** - Copy to your Hermes skills directory
2. **Install dependencies** - `pip install -r requirements.txt`
3. **Start learning** - Run the Python scripts or use via Hermes Agent
4. **Check examples** - See [EXAMPLES.md](EXAMPLES.md) for detailed usage

### For Contributors

1. **Fork the repository**
2. **Create a feature branch** - `git checkout -b feature/amazing-feature`
3. **Make your changes** - Follow our coding standards
4. **Test thoroughly** - Ensure everything works
5. **Submit a Pull Request** - We'll review it ASAP!

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines.
