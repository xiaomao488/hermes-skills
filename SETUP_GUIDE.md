## 🔧 需要手动完成的配置

### 1. 启用 GitHub Discussions

**步骤：**
1. 访问：https://github.com/xiaomao488/hermes-skills
2. 点击顶部的 **Settings** 标签
3. 在左侧菜单找到 **Features** 部分
4. 勾选 ✅ **Discussions**
5. 点击 **Set up discussions** 按钮
6. GitHub 会自动创建欢迎帖子

**为什么需要：**
- 让社区成员讨论想法和问题
- 提供 Q&A 区域
- 分享使用经验和最佳实践

---

### 2. 添加 Topics 标签

**步骤：**
1. 访问：https://github.com/xiaomao488/hermes-skills
2. 在仓库名称下方找到 **About** 部分（右侧）
3. 点击 ⚙️ 齿轮图标（设置）
4. 在 **Topics** 输入框中添加以下标签（按回车添加每个）：
   - `hermes-agent`
   - `documentation`
   - `github`
   - `learning`
   - `knowledge-base`
   - `ai-agent`
   - `automation`
   - `python`
   - `web-scraping`
5. 点击 **Save changes**

**为什么需要：**
- 让项目更容易被搜索到
- 帮助用户发现相关项目
- 提高项目可见性

---

### 3. 创建 GitHub Release

**步骤：**
1. 访问：https://github.com/xiaomao488/hermes-skills/releases
2. 点击 **Create a new release** 按钮
3. 在 **Choose a tag** 下拉框中选择 `v1.0.0`
4. 填写 Release 信息：

**Release title:**
```
v1.0.0 - 首次发布
```

**Release description:**
```markdown
# 🎉 Hermes Skills 首次发布

这是 hermes-skills 项目的第一个正式版本！

## ✨ 包含的技能

### 📚 Web Documentation Reader
从网页中提取和学习技术文档内容的技能。

**功能：**
- 智能提取网页正文内容
- 支持 Markdown 和 JSON 格式输出
- 自动提取代码块和链接
- 保存到 Obsidian 知识库

### 🔍 GitHub Project Learner
深入学习 GitHub 项目的完整内容。

**功能：**
- 提取 README、文档、代码结构
- 分析 Issues 和 Discussions
- 读取 Wiki 内容
- 支持批量学习多个项目

## 📦 安装

```bash
git clone https://github.com/xiaomao488/hermes-skills.git
cd hermes-skills
pip install -r requirements.txt
```
