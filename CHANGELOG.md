# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-05-27

### Added
- CDP troubleshooting guide for browser connection issues
- Quick setup checklist for new users
- Comprehensive setup guide for GitHub configuration
- Community contribution support documentation

### Changed
- Improved README with better structure and badges
- Enhanced documentation organization

## [0.1.0] - 2026-05-27

### Added
- Initial release of two core skills:
  - `web-documentation-reader` - Extract and learn from web documentation
  - `github-project-learner` - Deep learning of GitHub projects
- Python extraction scripts for both skills
- Comprehensive examples and usage documentation
- MIT License
- Contributing guidelines
- Requirements.txt for Python dependencies
- GitHub Actions workflow for CI/CD

### Features
- Web documentation extraction with Markdown support
- GitHub project analysis (README, docs, structure, issues)
- Obsidian-compatible knowledge base output
- Batch processing support
- Code example extraction
- Project metadata and statistics

### Documentation
- Detailed README with installation and usage instructions
- EXAMPLES.md with comprehensive API documentation
- CONTRIBUTING.md with contribution guidelines
- SETUP_GUIDE.md for complete setup instructions

## [0.0.1] - 2026-05-27

### Added
- Initial project structure
- Basic skill templates
- Repository setup

## Upgrade Guide

### From 0.1.0 to 0.2.0
No breaking changes. Simply pull the latest changes:
```bash
cd /path/to/hermes-skills
git pull origin main
```
